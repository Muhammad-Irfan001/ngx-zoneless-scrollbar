# ngx-zoneless-scrollbar Demo

This is a StackBlitz demo showcasing the `ngx-zoneless-scrollbar` package with different configurations.

## Features Demonstrated

- **Column 1**: Minimal content (scrollbar doesn't activate)
- **Column 2**: Default scrollbar styling
- **Column 3**: Dark theme with custom colors
- **Column 4**: Thin, minimalist scrollbar

## Running Locally

```bash
npm install
npm start
```
